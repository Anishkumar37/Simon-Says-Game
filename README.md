# Simon Says Game

A simple browser-based implementation of the classic "Simon Says" memory game.

## Preview

- Open the game in your browser to play: [simon says game/index.html](simon%20says%20game/index.html)

## How to run

- Double-click [simon says game/index.html](simon%20says%20game/index.html) to open in your default browser.
- Or serve the folder and open the page (useful for some browsers):

```bash
python -m http.server 8000
# then open http://localhost:8000/simon%20says%20game/index.html
```

## Controls

- Press any key to start the game.
- Click the colored tiles to repeat the sequence.

## Files

- [simon says game/index.html](simon%20says%20game/index.html) — main HTML file
- [simon says game/script.js](simon%20says%20game/script.js) — game logic
- [simon says game/style.css](simon%20says%20game/style.css) — styling

## Notes

- The game increases the level each round and shows "Game Over" on mistakes.
- Feel free to modify `script.js` to change timing, sounds, or add features.
